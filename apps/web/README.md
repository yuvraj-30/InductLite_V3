# InductLite Web App

See root README for full documentation.

## Development

```bash
# From root directory
npm install
npm run dev
```

## Database

```bash
# Generate Prisma client
npm run db:generate

# Create migration
npm run db:migrate:dev

# Apply migrations
npm run db:migrate

# Seed database
npm run db:seed

# Open Prisma Studio
npm run db:studio
```
