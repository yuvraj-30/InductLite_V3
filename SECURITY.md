# Security Policy

## Supported Versions

Only the latest production release is supported with security fixes.

## Reporting a Vulnerability

Please report security issues privately and **do not** open a public issue.

- Email: security@inductlite.example
- Include: steps to reproduce, impact, and affected endpoints/data.
- We will acknowledge receipt within 2 business days.

## Disclosure Process

1. Acknowledge and validate the report.
2. Assess severity and impact.
3. Develop and test a fix.
4. Coordinate disclosure with the reporter.

## Security Contacts

Primary: security@inductlite.example
